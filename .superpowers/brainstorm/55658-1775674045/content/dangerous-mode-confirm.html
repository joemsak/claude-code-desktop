Key Decisions

  • Both buttons always visible — "Launch Dangerous" (red) and "Launch Normal" (neutral). Declining isn't cancel, it's "launch safely instead"
  • "Change default mode in Settings" link always present regardless of current setting
  • When default is ON — extra note "This is your default — launched via Cmd+T" so user understands why they're seeing the prompt on a regular Cmd+T
  • Escape key dismisses modal → launches normal (same as clicking "Launch Normal")
